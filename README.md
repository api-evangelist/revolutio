# Revolutio (revolutio)

Revolutio provides smart structural engineering software and a Hazard API (formerly CHECKWIND.API) that delivers site-specific wind, snow, ice, and seismic load parameters for structural engineering and construction projects. The API is trusted by over 750 companies and 5000 users worldwide across construction, fabrication, power distribution, signage, structural engineering, and telecommunications. It covers 40+ countries and supports standards including AS/NZS 1170.2, AS 4055, and ASCE 7.

## APIs

### Revolutio Hazard API

The Revolutio Hazard API (formerly CHECKWIND.API) v4 provides site-specific wind, snow, ice, and seismic hazard parameters for structural engineering. Given a latitude/longitude and optional structure height, the API returns design parameters per AS/NZS 1170.2, AS 4055, ASCE 7, and other international standards. Uses a credit-based subscription model where only successful requests consume credits.

- **Human URL:** [https://www.revolutio.com.au/software/hazardapi/](https://www.revolutio.com.au/software/hazardapi/)
